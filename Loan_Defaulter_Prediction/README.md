# Loan Defaulter Prediction Model

## Overview
This project develops a classification model to predict loan defaulters using banking and financial data. It aims to enhance risk assessment and support data-driven decision-making in credit evaluations.

## Features
- Performed Exploratory Data Analysis (EDA) to understand the dataset and visualize patterns.
- Conducted statistical hypothesis testing to identify key factors influencing loan default.
- Implemented machine learning algorithms including Logistic Regression, Decision Trees, and Random Forest.
- Achieved an overall accuracy of 85–90%, demonstrating strong predictive performance.

## Technologies Used
- Python for data analysis, modeling, and visualization.
- MS Excel for initial data exploration and preprocessing.
- Machine learning libraries (e.g., scikit-learn) for algorithm implementation.

## Usage
- Load the dataset and preprocess it.
- Run EDA scripts to understand feature distributions.
- Train and evaluate ML models using provided code.
- Use the model to predict loan default probabilities on new data.

## Outcome
The model improves credit risk assessment accuracy, helping financial institutions make more reliable lending decisions based on data insights.
