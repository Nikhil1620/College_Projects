# PDF-ChatBot
GenAI‑Powered PDF Assistant is a web application that uses Google Generative AI, LangChain, and ChromaDB to let users upload PDFs and query them conversationally. It extracts, intelligently chunks, embeds, and stores document content to support fast, contextual Q&A via the Gemini‑2.0 model. This tool streamlines information retrieval from dense documents, making it ideal for researchers, educators, legal professionals, and analysts.
