# E-Commerce Sales Dashboard

This repository contains an interactive and visually appealing E-Commerce Sales Dashboard designed to analyze and monitor key sales metrics across various states, customers, product categories, and more.

## Dashboard Overview

The dashboard provides real-time insights into sales performance, customer spending patterns, and product-wise analysis. It is ideal for business analysts, data scientists, and stakeholders looking to quickly understand business trends and make data-driven decisions.

### Key Features

- **Total Sales (438K):** Displays the overall sum of sales amount.
- **Profit (37K):** Summarizes total profit generated.
- **Quantity Sold (5,615):** Shows the total quantity of items sold.
- **Average Order Value (AOV - 121K):** Presents the average value per order.

### Visualizations

- **Amount by State:** Horizontal bar chart comparing sales amounts across major states.
- **Profit by Month:** Bar chart showing monthly profit/loss trends throughout the year.
- **Top Customers:** Highlights customers with the highest purchase amounts.
- **Product Category Breakdown:** Pie chart illustrating quantity share by product category (Clothing, Electronics, Furniture).
- **Payment Modes:** Pie chart summarizing order share by payment mode (COD, UPI, Debit, Credit, EMI).
- **Profit by Sub-Category:** Bar chart with sub-category level profit breakdown (Printers, Bookcases, Saree, Accessories, Tables).

### How to Use

1. Clone this repository.
2. Open the dashboard file on your local machine using compatible BI tools (e.g., Power BI, Tableau, Streamlit, or custom web frameworks).
3. Filter and interact with the visualizations for detailed analysis (filters by quarter and state available).
